# Task 1: Data Preprocessing

## Objective
Prepare the Titanic dataset for analysis by cleaning and preprocessing it.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Google Colab

## Steps Performed
1. Loaded the Titanic dataset.
2. Checked missing values.
3. Filled missing values in Age.
4. Encoded the Sex column.
5. Normalized Age and Fare.
6. Saved the cleaned dataset.

## Files
- titanic.csv
- titanic_cleaned.csv
- Task1_Data_Preprocessing.ipynb

## Outcome
Successfully cleaned and prepared the dataset for further analysis.
